## ⭐주요 기능
- 카카오 맵 api를 이용한 지도 기능들(게시판 연동)
- 실시간 소통 가능한 채팅 기능(친구 목록, 채팅 목록)
- CCTV 위치 정보 제공 기능
- 사용자 매너온도
- 관심사 설정
